# パズル 3001 - 3x3 サンプル

このフォルダには以下のタイル画像が必要です:

- `tile_1.png` ~ `tile_9.png`

各タイル画像は正方形で、統一されたサイズにしてください。
推奨サイズ: 200x200px 以上

## パズルID命名規則

パズルIDの先頭の数字がグリッドサイズを表します:

- `3xxx` - 3x3パズル (9タイル)
- `4xxx` - 4x4パズル (16タイル)
- `5xxx` - 5x5パズル (25タイル)

例:
- `3001` = 3x3の1番目
- `4001` = 4x4の1番目
- `5001` = 5x5の1番目
